# 🇨🇱 Sistema de Gestión de Aduanas - Paso Fronterizo Los Libertadores

Este repositorio contiene la estructura oficial de ingeniería de software y la documentación técnica para el **Proyecto de Gestión de Aduanas**, diseñado para centralizar, digitalizar y optimizar los flujos de control migratorio, aduanero y de sanidad en los pasos fronterizos de Chile.

---

##  Historial de Versiones e Hitos del Proyecto

La evolución de este proyecto se encuentra estrictamente alineada con la **Carta Gantt y la EDT** definidas en la planificación inicial:

* **v1.0 (Hito de Planificación):** Definición de las reglas del caso de estudio de Aduanas, organización del equipo y análisis preliminar de usuarios.
* **v1.1 (Hito de Ingeniería de Requisitos):** Formalización de las matrices de Requisitos Funcionales y No Funcionales bajo el estándar **IEEE 830**.
* **v1.2 (Hito de Arquitectura, Calidad y Cierre):** Estructuración técnica mediante el **DAS (Documento de Arquitectura de Software)**, diagramas UML completos y auditoría de interfaz mediante la **Evaluación Heurística de Nielsen**.

---

##  Estructura del Repositorio

El proyecto se divide en las siguientes fases metodológicas:

1.  **`01-analisis-y-planificacion/`**: Contiene el caso de estudio base, la planificación temporal (Carta Gantt/EDT) y apuntes conceptuales de usuarios.
2.  **`02-ingenieria-de-requisitos/`**: Matriz de trazabilidad y especificación de requisitos funcionales y no funcionales.
3.  **`03-arquitectura-y-calidad/`**: Documento de Arquitectura de Software (DAS v1.2) junto con la planilla de evaluación de usabilidad.
4.  **`04-diseno-ui-y-uml/`**:
    * `diagramas-uml/`: Vistas físicas, lógicas, diagramas de componentes y de paquetes que sostienen el sistema.
    * `mockups-canva/`: Capturas del prototipo de alta fidelidad para los diferentes módulos (Pasajeros, Funcionarios, Administradores y Analistas).
